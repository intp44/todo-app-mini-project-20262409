# todo-app-mini-poject-20262409
서비스 바로가기
배포 주소: https://todo-app-mini-project-20262409.vercel.app

주요 기능
실시간 알림: 사용자가 설정한 초 단위 시간에 맞춰 브라우저 알람 제공

할 일 관리(CRUD): 할 일 추가, 목록 조회, 삭제 기능 연동 (MongoDB)

반응형 다크 모드 UI: 눈이 편안한 세련된 다크 모드 인터페이스 적용

프로젝트 정보 가이드: 사이드바를 통해 프로젝트의 상세 설명 및 기술 스택 확인 가능

기술 스택
Frontend
React: 컴포넌트 기반 UI 구현 및 상태 관리

CSS3: 다크 모드 스타일링 및 레이아웃 최적화

Backend
Node.js & Express: RESTful API 서버 구축

MongoDB Atlas: 클라우드 NoSQL 데이터베이스 연동

Mongoose: 데이터베이스 스키마 정의 및 모델링

Deployment & DevOps
Vercel: 프론트엔드 및 서버리스 함수 배포

Git & GitHub: 버전 관리 및 협업 환경 구축

설치 및 실행 방법
저장소 클론
Bash
git clone https://github.com/intp44/todo-app-mini-project-20262409.git
cd todo-app-mini-project-20262409
2백엔드 설정
backend 폴더 내에 .env 파일을 생성하고 MongoDB 연결 주소를 입력합니다.

Plaintext
MONGODB_URI=your_mongodb_connection_string
Bash
cd backend
npm install
npm start
프론트엔드 설정
Bash
cd ../frontend
npm install
npm start
